## 这是什么

这是一个跑在浏览器里的 **3D粒子星系**，用摄像头追踪你的手部动作来控制星系的旋转、缩放和形态切换。

不需要安装任何插件，不需要下载任何APP，打开网页就能用。

核心就两个东西：

- **Three.js** — 负责渲染14000个粒子组成的3D场景
- **MediaPipe Hands** — 负责实时追踪手部21个关键点

## 技术实现

### 粒子系统

用 Three.js 的 `BufferGeometry` 创建了 14000 个粒子，每个粒子有独立的位置、颜色和大小。不同形态有不同的生成函数，比如星系形态用对数螺线方程：

```
r = a * e^(b * θ)
```

粒子沿旋臂分布，加上随机偏移模拟真实星系的弥散感。黑洞形态则分了四层环（内环亮白热、中环橙红、外环暖黄、最外环蓝），加上 10% 粒子做相对论性喷流、5% 做光晕。

### 手势识别算法

MediaPipe Hands 返回 21 个手部关键点的 3D 坐标。我用其中几个关键点做了四种手势判断：

**手掌位置（控制旋转）：**
```
d.x += ((0.5 - lm[8].x) * 2 - d.x) * 0.4;
```
用食指尖（关键点 8）的坐标，加上 0.4 的平滑系数做低通滤波，避免手抖导致画面剧烈晃动。

**张开/握拳判断（控制缩放）：**
- 4 根手指：比较每个手指的指尖（tips: 8,12,16,20）和指间关节（pips: 6,10,14,18）的 Y 坐标，指尖在上就算伸展
- 拇指：比较拇指尖（关键点 4）和食指根（关键点 2）的 X 距离，大于 0.04 算伸展
- 伸展 ≥ 4 根 → 张开手掌（expansion = 1.5，星系放大）
- 食指尖到手掌根（关键点 0）距离 < 0.25 → 握拳（expansion = 0.3，星系收缩）

**捏合判断（切换形态）：**
```
dp = sqrt((lm[8].x - lm[4].x)² + (lm[8].y - lm[4].y)² + (lm[8].z - lm[4].z)²)
isPinching = dp < 0.05 && ext >= 3
```
拇指尖和食指尖的 3D 欧氏距离小于 0.05，且至少 3 根手指伸展（排除握拳误判）。加了 600ms 防抖，避免连续触发。

**双手靠近（触发爱心）：**
两只手同时检测到时，计算双手距离。距离 < 0.25 且持续 40 帧（约 1.3 秒）才触发爱心形态，避免误触。

### 心跳动画

爱心形态有独立的心跳节拍：

```
beatPhase = (time * 4) % (2π)
beatScale = beatPhase < 0.3 ? 1.0 + sin(beatPhase / 0.3 * π) * 0.15 : 1.0
```

频率约 0.63Hz（每秒跳 4 次周期中的 1 次），每次跳动膨胀 15%，持续 0.3 个周期后回弹。中心辉光的透明度也跟着心跳同步脉动（0.06 ± 0.04）。

### 粒子过渡动画

形态切换不是瞬间替换，14000 个粒子从当前位置 lerp 到新位置：

```
posArr[ix] += (targetX - posArr[ix]) * lerpSpd + noise
```

爱心形态 lerp 速度 0.08（更快的弹性感），其他形态 0.04。加了正弦噪声 `sin(time * 0.8 + i * 0.1) * 0.015` 让粒子有微弱的呼吸感，不会完全静止。

相机旋转和缩放也做了平滑插值（系数 0.05-0.06），手势移动时画面不会突兀跳变。

### 陨石坠落

陨石形态是唯一不做 lerp 过渡的——粒子从 Y=18 坠落到 Y=-16，到达地面后随机重置到顶部。手势控制 X/Z 方向的漂移，模拟「用手引导流星方向」的感觉。颜色从高空的冷蓝渐变到低空的火焰橙红。

### 性能优化

14000 个粒子每一帧都要更新位置，性能是关键：

- 用 `BufferGeometry` 而不是 `Geometry`，直接操作 Float32Array
- 形态切换时用 lerp 插值，不用 tween 库
- MediaPipe 模型文件自托管 + preload，减少加载时间
- `.tflite` 文件重命名为 `.bin` 避免 MIME 类型问题
- 相机分辨率降为 320×240（手势追踪够用，省一半算力）

## 部署

整个项目是一个纯静态页面，部署在 Cloudflare Pages 上。MediaPipe 的 WASM 文件、Three.js、手部追踪模型全部自托管，不依赖任何外部 CDN。

源文件结构：

```
galaxy/
├── index.html          # 主页面（HTML + CSS + JS 全在一个文件）
└── mediapipe/
    ├── hands.js        # MediaPipe Hands API
    ├── *.wasm          # WASM 运行时
    ├── *.bin           # 手部追踪模型
    └── three.min.js    # Three.js
```

没有构建工具，没有 npm，没有打包——一个 HTML 文件加一堆静态资源，拖到 Cloudflare Pages 就上线了。
