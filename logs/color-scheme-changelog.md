# 配色优化修改日志

**日期:** 2026-06-15
**目标:** 提亮配色，增加生机感，告别沉闷暗色调
**状态:** 预览阶段（未应用到正式博客）

---

## 修改清单

| # | 修改项 | 原始值 | 新值 | 说明 |
|---|--------|--------|------|------|
| 1 | 亮色背景 | `#ffffff` 纯白 | `#faf9f7` 暖白 | 更温暖，不刺眼 |
| 2 | 暗色背景 | `rgb(29,30,32)` 冷灰 | `#1a1b1e` 柔和深灰 | 更舒适 |
| 3 | 强调色 | 无 | `#14b8a6` / `#2dd4bf` 青绿 | 新增生命力 |
| 4 | 链接色 | 默认蓝 | `#0d9488` 青绿 | 统一色调 |
| 5 | 卡片悬停 | 无 | 上浮2px + 左边框动画 | 增加交互感 |
| 6 | 标签样式 | 默认 | 药丸圆角 + 悬停变色 | 更现代 |
| 7 | 代码块 | 直角 | 8px圆角 | 更柔和 |
| 8 | 头像 | 无特效 | 3px青绿边框 + 光环 + 悬停缩放 | 更精致 |
| 9 | 选中文字 | 默认蓝色 | 青绿高亮 | 统一 |
| 10 | 滚动条 | 系统默认 | 自定义8px圆角 | 更精致 |

## 预览文件

- `preview-color-scheme.html` — 独立预览，不影响正式博客
- 包含亮色/暗色两种模式切换
- 包含颜色对比色卡

## 回滚方案

直接删除预览文件即可，正式博客未做任何修改：
```bash
rm preview-color-scheme.html
```

## 应用方案（确认后执行）

如果效果满意，将优化后的 CSS 添加到 `assets/css/extended/custom.css`：
```bash
# 1. 备份当前 table.css
cp assets/css/extended/table.css assets/css/extended/table.css.bak

# 2. 创建 custom.css（包含所有优化）
# 内容见预览文件中的 <style> 标签

# 3. 提交并部署
git add -A && git commit -m "style: brighten color scheme, add accent colors"
git push
```
